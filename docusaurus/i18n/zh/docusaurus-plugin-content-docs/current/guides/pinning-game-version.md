# 固定游戏版本

:::warning
降级到较低的游戏版本是可能的，但目前无法确定这将对现有存档产生什么影响。

**请自行承担风险！**
:::

如果设置了 **TARGET_MANIFEST_ID** 环境变量，将会锁定服务器版本到特定的清单（manifest）。
清单对应发布日期/更新版本。可以使用 SteamCMD 或类似 [SteamDB](https://steamdb.info/depot/2857201/manifests/) 的网站找到清单。
